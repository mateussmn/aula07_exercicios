# Aula 07 - Exercícios em Python

Repositório contendo os exercícios desenvolvidos durante a Aula 07 de Python, com foco em funções, listas, tipagem estática e manipulação de dados.

## 📚 Conteúdo

Os exercícios abordam conceitos fundamentais da linguagem Python, como:

- Criação de funções
- Tipagem com `typing`
- Manipulação de listas
- Cálculo de média
- Filtragem de valores
- Estruturas de repetição (`for`)
- Estruturas condicionais (`if`)
- Retorno de valores em funções
